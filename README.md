# Online Food Ordering System

## Description
This project is an Online Food Ordering System developed in Java. It allows users to browse a menu, add items to a cart, and place orders using different payment methods.

## Features
* **User & Customer Classes:** Manages user details and cart.
* **Menu System:** View items and prices.
* **Polymorphism:** Supports payment via Credit Card or Cash.
* **OOP Principles:** Uses Encapsulation, Inheritance, and Interfaces.
